# 關於
這是一個興趣使然的課餘項目，意在搜集和整理香港自然史、文化史、藝術史等各類研究文獻，並製作成資料索引集，予芸芸學子使用。

資料主要來源（排名不分先後）：
- 香港公共圖書館多媒體資訊系統：[香港舊報紙](https://mmis.hkpl.gov.hk/zh/old-hk-collection)
- 香港古物古蹟辦事處參考圖書館：[網上圖書館目錄](https://lms.amo.gov.hk/hk)

項目成員：
- 北京大學考古系在讀博士生阿聰
- 香港中文大學歷史系在讀本科生鄺某
